# My vim configuration
## Prerequisite
- Vim 8 and above

## Configuration

```
git clone https://github.com/Msiciots/vim-conf
mv vim-conf/.vim ~/
mv vim-conf/.vimrc ~/
```
Open vim and execute this command to install plugins.
```
:PlugInstall
```
## Shortcut key
- F1 : NERDTreeToggle
- F2 : Toggle of relative line number
